## 📌 Overview

A desktop implementation of the classic **Tic-Tac-Toe (XO)** game built using **C# and Windows Forms**.  
This project focuses on building clean game logic and handling user interaction in a structured way.

---

## 🧠 Core Features

- 🎯 Turn-based gameplay (Player 1 vs Player 2)  
- ✅ Real-time win detection  
- 🔄 Draw condition handling  
- 🎨 Highlight winning combination  
- 🔁 Restart game anytime  
- 🖱️ Interactive UI  

---

## ⚙️ How It Works

The game tracks:

- Current player turn  
- Number of moves  
- Game state (Win / Draw / In Progress)

After each move:
1. The button updates (X or O)
2. Game state changes
3. Win conditions are checked:
   - Rows
   - Columns
   - Diagonals

---

## 🖥️ UI Design

- Built using **Windows Forms**
- 3×3 grid (buttons)
- Custom grid drawn using `Paint` event
- Image-based interaction
- Visual feedback for winner
